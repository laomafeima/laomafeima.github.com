PHP 导入并运行 php 脚本文件

在开发的过程中经常遇到导入并运行 php 脚本文件。
这里封装函数可以直接导入并运行脚本

int include_file_scripts(char *file_name) {
 
    zend_file_handle fh;
    fh.filename = file_name;
    fh.opened_path = NULL;
    fh.free_filename = 0;
    fh.type = ZEND_HANDLE_FILENAME;
    zend_execute_scripts(ZEND_INCLUDE TSRMLS_CC, NULL, 1, &fh);
    return SUCCESS;
}

调用的时候直接传入 php 脚本文件的路径即可。
