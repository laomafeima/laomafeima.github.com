二分法查找算法

```
function dichotomie($arr,$v)
{
    $start = 0;
    $end = count($arr) - 1;

    while($start <= $end)
    {
        $index = intval(($start + $end) / 2);
        if($v < $arr[$index])
        {
            $end = $index - 1;
        }
        elseif($v > $arr[$index])
        {
            $start = $index + 1;
        }
        else
        {
            return $index;
        }
    }
    return -1;
}
```
